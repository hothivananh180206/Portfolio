About Me

Xin chào, tôi là Vân Anh

Tôi là sinh viên năm 2 ngành Công nghệ Thông tin tại Cao đẳng Công nghệ Thủ Đức (CĐCNTD), chuyên ngành Phát triển Web. Tôi bắt đầu học lập trình web từ những kiến thức nền tảng như HTML, CSS và JavaScript, sau đó dần phát triển kỹ năng xây dựng giao diện và ứng dụng web có tính tương tác. Hiện tại tôi tập trung vào lĩnh vực phát triển web (Web Development), nơi tôi có thể kết hợp giữa tư duy lập trình và thiết kế giao diện để tạo ra các sản phẩm thực tế trên trình duyệt.

Tôi học thông qua việc kết hợp giữa lý thuyết và thực hành tại trường. Bên cạnh các môn học, tôi tự đọc tài liệu và tham gia các đồ án nhóm tại trường để làm quen với môi trường làm việc thực tế, cũng như sử dụng Git và GitHub để phối hợp và quản lý mã nguồn cùng nhóm.

  • Xây dựng trang web responsive với HTML, CSS, JavaScript
  • Làm việc nhóm qua Git & GitHub trong các đồ án trường
  • Đang tìm hiểu thêm về animation và trải nghiệm người dùng

Hệ Thống Quản Lý Kho Hàng

Đồ án nhóm 2 người, trong đó tôi phụ trách phát triển giao diện frontend và các chức năng chính của hệ thống. Cụ thể, tôi xây dựng các module quản lý sản phẩm, nhà cung cấp, khách hàng, cùng với chức năng nhập – xuất kho và giao diện quản lý tổng thể. Trong quá trình làm việc nhóm, tôi sử dụng Git để quản lý mã nguồn, thực hành làm việc với nhánh (branch), merge code và xử lý xung đột (conflict) khi tích hợp các phần của dự án.
